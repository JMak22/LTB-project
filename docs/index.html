LTB Annual Report Data Explorer

Public-facing exploration interface for cleaned Ontario Landlord and Tenant Board annual report data (1998-2025).

Last updated: May 27, 2026
Last database rebuild: May 28, 2026

About this project

This project aggregates and structures publicly available annual report data related to the Ontario Rental Housing Tribunal, the Landlord and Tenant Board and Tribunals Ontario.

The goal is to make institutional data more accessible to researchers, journalists, tenant advocates, policymakers and the public.

Start here

Open Datasette Explorer

Use the data explorer to search tables, filter records, inspect metrics, and download data for your own analysis.

What can I explore?

  • Operational metrics: applications received, resolved and outstanding.
  • Financial metrics: reported revenue, expenses, and fee-related data, for the years when they were reported in Annual Reports.
  • Staffing metrics: adjudicator/member counts. Data came directly from Annual Reports where given, or from `member_counts_by_file.csv`
  • Regional metrics: application breakdown by region.
  • Application type metrics: application categories such as L1, T1 and other filing types. Note that application type names have changed over the years, the database does not reflect those changes for the sake of our sanity.

Questions to try

  • In which years did received applications exceed resolved applications?
  • How did unresolved cases change over time?
  • Which application types appear to grow or shrink across report years?

Limitations

The data comes from annual reports which changed format and level of detail over time. Some metrics are unavailable, renamed, aggregated differently, or reported inconsistently across years. Use this explorer as a structured reference, not Complete God's Own Truth About the LTB.
